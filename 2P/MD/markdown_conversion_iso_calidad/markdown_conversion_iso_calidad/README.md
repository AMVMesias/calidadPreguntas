# Conversión a Markdown - Documentos de estándares de calidad de software

Este ZIP contiene una conversión a Markdown por cada documento y por cada enlace entregado.

## Archivos incluidos

1. `01_ISOIEC_9241.md` - Presentación PDF sobre ISO/IEC 9241 y calidad en uso.
2. `02_Actividad_ISO9126.md` - Informe académico sobre ISO/IEC 9126.
3. `03_Estandares_Calidad_Proceso_ISO9001_ISO29110.md` - Presentación PDF sobre calidad del proceso, ISO 9001 e ISO/IEC 29110.
4. `04_Estandares_Calidad_Producto_Software_ISO9126_CISQ.md` - Presentación PPTX sobre calidad del producto de software, ISO/IEC 9126 y CISQ.
5. `05_Link_ISO25000_ISOIEC25010.md` - Contenido del portal ISO 25000 sobre ISO/IEC 25010.
6. `06_LinkParte2_ISO_Oficial_ISOIEC25010_2023.md` - Resumen de la página oficial ISO para ISO/IEC 25010:2023.

## Carpeta de assets

La carpeta `assets/` contiene una imagen renderizada por cada página o diapositiva. Esto se incluyó para conservar la información visual: tablas, diagramas, gráficos, capturas e infografías que pueden no quedar perfectamente representadas como texto Markdown.

## Nota de conversión

No se aplicó OCR masivo. La conversión combina el texto extraíble de los documentos con imágenes renderizadas de cada página/diapositiva, más notas visuales en páginas donde el contenido principal estaba representado como diagrama o gráfico.
